SocialGeoMovie ADMIN Endpoint's

ADMIN: Populate Movies in Neo4J database (import Movie's Data)

  • URL: http://localhost:8080/aw2017/rest/db/movies

GET

  • URL Params: No Params

  • Success Response:

  • Code: 200
    Content: {"movies":"#n","status":"OK"}

    • #n: number of movies retrieved

  • Error Response:

  • Code: 404 NOT FOUND
    Content: { error 404 }

OR

  • Code: 401 UNAUTHORIZED
    Content: { error : "You are unauthorized to make this request." }

  • Sample Call:

javascript $.ajax({ url: "/aw2017/rest/db/movies", dataType: "json", type : "GET", success : function(r) { console.log(r); } });
