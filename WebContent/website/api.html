SocialGeoMovie Endpoint's

Show People

Returns json data of all Person's in database.

  • URL: http://localhost:8080/aw2017/rest/person

GET

  • URL Params: No Params

  • Success Response:

  • Code: 200
    Content: [{"name":"Henardo Rodriguez","id":852544}, ... ]

  • Error Response:

  • Code: 404 NOT FOUND
    Content: { error : "User doesn't exist" }

    • OR

    • Sample Call:

    javascript $.ajax({ url: "/aw2017/rest/person", dataType: "json", type : "GET", success : function(r) { console.log(r); } });
